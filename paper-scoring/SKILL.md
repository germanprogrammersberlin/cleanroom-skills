# Paper Scoring

Score the paper dimensionally and give an overall recommendation.

## 5 Dimensions (weighted)

| Dimension | Weight | Scale |
|-----------|--------|-------|
| Originality | 20% | 0-100 |
| Methodology | 25% | 0-100 |
| Evidence | 25% | 0-100 |
| Coherence | 15% | 0-100 |
| Writing | 15% | 0-100 |

**Total** = Originality×0.20 + Methodology×0.25 + Evidence×0.25 + Coherence×0.15 + Writing×0.15

## Decision

- >= 80: Accept
- 65-79: Minor Revision
- 50-64: Major Revision
- < 50: Reject

## Scale Definitions

**Originality (20%):**
- 90-100: New framework, opens new direction
- 75-89: New method OR new application
- 60-74: Extends existing with new data
- < 60: Replication with minimal variation

**Methodology (25%):**
- 90-100: Design perfectly matched to research question
- 75-89: Solid design with minor gaps
- 60-74: Acceptable but justification missing
- < 60: Fundamental design flaws

**Evidence (25%):**
- 90-100: >40 sources, multi-method triangulation
- 75-89: 25-40 sources, sufficient evidence
- 60-74: 15-25 sources, core claims supported
- < 60: Under-evidenced, unsupported claims

**Coherence (15%):**
- 90-100: Crystal-clear logical flow
- 75-89: Clear flow with minor gaps
- 60-74: Main argument visible but sections feel disconnected
- < 60: No coherent argument

**Writing (15%):**
- 90-100: Professional prose, zero grammar errors
- 75-89: Good writing, few issues
- 60-74: Acceptable, room for improvement
- < 60: Below journal standards

## Review Output

Write a review containing:
1. Scores table with justification per dimension
2. Decision (Accept/Minor/Major/Reject)
3. Summary (3-5 sentences)
4. Strengths (numbered)
5. Weaknesses (numbered)
6. Questions for Authors
7. Missing References
8. Recommendation (detailed justification)

For each actionable weakness, create a separate finding.
